ARPABET 是Arpasing式中所使用的音标，不区分大小写（本来是为美式英语开发的）
※通常为每一音素空格＆每一单词加点“.”的标记样式，不过在此表中未体现
※通常用0~3的数字表示重音（AA0, AA1, AA2等），不过UTAU的歌詞=别名，重音没有关系，预测时可以无视

X-SAMPA 是Delta式中所使用的音标，区分大小写（X-SAMPA的前身SAMPA是本来为英式英语开发的） ※SAMPA的 −对应欧洲各国以外的语言的− 扩展版即为X-SAMPA

【研究】/ʌ/和/ə/
虽然音标不同（ʌ,ə），但两者只是发声强弱有差异，本质上是一样的东西
→有重音时为“啊”的发音（/ʌ/）
→无重音时为弱央元音（/ə/）
因此ARPABET的ah即为，有重音时为 ah（/ʌ/），无重音时为 ax（/ə/）
【研究】/ɪ/与/iː/、/ʊ/与/uː/发声不同 （※粉色字）
IPA中所说的/ɪ/与/iː/、/ʊ/与/uː/ 分别为不同的音 它们不仅仅是长度的不同 存在发音的不同，与区分单词的含义有关联
※歌唱中元音的长度就作为音符的长度，因此发声的差异就被用来区分单词含义（的差异）

 【建议】无论如何既然/ɪ/与/iː/（或者说是/ɪ/与/i/？？）是不同的发音，希望能够区分发声＆别名
※偶尔也会有没有区分/ɪ/与/i/（X-SAMPA中就是[I]与[i]）的音源 所以要是这么做的话是不成的

  • Cz是Cz式中所使用的音素和音节标记 ※除Cz以外中以音素组合表示音节时，在表中以灰色文本表示
  • 特别地，单元音中a系元音标为红色粗体 ※并且X-SAMPA, Cz,VOCALOID分别完全没有相似性
  • 特别地，长元音中与短元音的/ɪ,ʊ/发声不同分别以粉色粗体的/iː,uː/标记
  • 带勾的IPA基本分别为[ɝ = ɜr, ɚ = ər, ɑ˞ = ɑr, ɔ˞ = ɔr]，省略
  • 下均省略 ⇒ 三元音、通常的辅音（b、d、k等）、日语特有的发音（辅音/元音）

VOCALOID是有额外补充的。总的来说和X-SAMPA是相关关系（大抵相同，细微之处不同）。
要详细点说，UTAU的歌词中使用的别名，可以通过这些音素组合实现。
